## 性能优化建议
1. 图表使用防抖处理窗口大小变化
2. 移动端隐藏不必要的滚动条
3. 使用 CSS transform 进行视图切换动画
4. 卡片视图使用 box-shadow 轻量化

## 已知问题和限制
1. ECharts 图表在极小屏幕（<360px）可能需要进一步调整
2. 复杂表单在移动端可能需要分步骤显示
3. 批量操作在移动端体验有限，建议引导用户使用 PC 端
